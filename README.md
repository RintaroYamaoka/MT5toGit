MT5toGit

本リポジトリは個人制作物（個人開発）です。
業務コードや機密情報は含まれていません。

MetaTrader（EA開発用エディタ）がUTF-16でソースコードを保存するため、
Gitでバックアップ・差分管理しやすいように、リポジトリ全体をUTF-8へ変換してコピーするツールです。


背景

  MT5のEA開発では、ソースコード（.mq5 / .mqh）がUTF-16で保存されることがあります
  UTF-16のままだと、Git上の差分表示や外部ツールで扱いづらいケースがあります
  本ツールは、バックアップ用途として、対象フォルダをUTF-8へ変換した構成で別フォルダに出力します


できること

  指定したフォルダ配下のファイルを収集して別フォルダへコピーする
  対象拡張子（.mq5 / .mqh）をUTF-16からUTF-8へ変換して保存する
  元フォルダは変更せず、変換後のフォルダを生成する


使い方

PySide6をインストールします

```bash
pip install PySide6

アプリを起動します
python main.py
